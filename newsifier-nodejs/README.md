## Running Project
These are the commands you should execute one by one to run the project:
- git init
- git clone https://github.com/alishaheen1988/newsifier-nodejs.git
- cd .\newsifier-nodejs\
- npm install
- npx tsc
- node dist/app.js
### then the app will run on port 3000
